#弹性盒布局(flex布局)
display:flex ;
flex:1 默认继承父元素高度; auto(自适应)
##一 、主侧轴切换
```CSS
<style>
    /*定义一个弹性盒,把.box当作一个容器;*/
        .box{
            height: 600px;
            display: flex;
            /*1.改变主轴方向*/
                /*flex-direction:column   */
            /*2.换行*/
                /*flex-wrap: wrap;*/
            /*3.改变主侧轴方向且换行与不换行*/
                /*flex-flow:column wrap;*/
        }
        .box div{
            /*flex: 1*/
            width: 80%
        }
        .red{
            background-color: red;
        }
        .blue{
            background-color: blue;
        }
        .green{
            background-color: green;
        }
    </style>
```
## 二 、主轴对齐
```CSS
<style>
        .box{
            height: 600px;
            background-color: #ccc;
            display: flex;
            /*
            justify-content:改变主轴方向下的子项目对齐方式;
            默认:左端对齐:flex-start;
            右端对齐:flex-end;
            居中:flex-center;
            平分中间空白部分:space-between;
            平分,两端保留一半的空间:space-around
            */
            justify-content: space-around;
        }
        .box div{
            width: 100px;
            height: 100px;
            border:10px solid red;
        }
    </style>
```
##三 侧轴对齐
###单行 align-items

```CSS
<style>
    /*定义一个弹性盒,把.box当作一个容器;*/
        .box{
            height: 600px;
            display: flex;
            border: 10px solid #ccc;
            /*
            align-items伸缩项目行在侧轴上的对齐方式（单行）
                flex-start：伸缩项目在侧轴起点边的外边距 紧靠住 该行在侧轴起始边；
                flex-end：伸缩项目在侧轴终点边的外边距 紧靠住 该行在侧轴终点边；
                center：伸缩项目的外边距盒 在该行的侧轴上居中放置；
                baseline：伸缩项目根据伸缩项目的基线对齐；
                stretch：伸缩项目拉伸填充整个伸缩容器。

            align-self：主要用来设置单独伸缩项目在侧轴的对齐方式，可以覆盖align-items。
                flex-start：各行向伸缩容器的起点位置堆叠；
                flex-end：各行向伸缩容器的结束位置堆叠；
                center：各行向伸缩容器的中间位置堆叠;
                stretch：各行将伸展以占用额外空间;
                space-between：各行在伸缩容器中平均分布；
                space-around：各行在伸缩容器中平均分布，两端保留一半的空间；
            */
            align-items:center;
        }
        .box div{
            flex: 1;
        }
        .red{
            background-color: red;
        }
        .blue{
            background-color: blue;
        }
        .green{
            background-color: green;
        }
    </style>
```
###多行 aligh-content
与justify-content是对应的;
```CSS
<style>
    /*定义一个弹性盒,把.box当作一个容器;*/ 
    .box {
        height: 600px;
        display: flex;
        border: 10px solid #ccc;
        flex-wrap: wrap;
        align-content: flex-end;
        /*
        align-content
            flex-start：各行向伸缩容器的起点位置堆叠；
            flex-end：各行向伸缩容器的结束位置堆叠；
            center：各行向伸缩容器的中间位置堆叠;
            stretch：各行将伸展以占用额外空间;
            space-between：各行在伸缩容器中平均分布；
            space-around：各行在伸缩容器中平均分布，两端保留一半的空间；
        */
    }
    
    .box div {
        width: 60%;
        height: 100px;
    }
    
    .red {
        background-color: red;
    }

    .blue {
        background-color: blue;
    }
    
    .green {
        background-color: green;
    }
    </style>
```
#四 改变伸缩项目的顺序;
```CSS
    <style>
        .box{
            height: 600px;
            display: flex;
            flex-wrap: wrap;
            border:10px solid #ccc;
        }
        .box div{
            width: 50%;
        }
        .red{
            /*order:默认按照标准流的顺序进行排放。*/
            background: red;
            /*order改变伸缩项目的顺序*/
                /*没写order属性的子项目在写了的子项目前面*/
                /*写了order属性的子项目按照属性值大小排放*/
            order:3;
        }
        .blue{
            background: blue;
            order:1;
        }
        .green{
            background: green;
            order:2;
        }

    </style>
```
